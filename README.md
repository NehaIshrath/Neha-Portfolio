# 🌐 Neha Ishrath | Portfolio Website

A modern and responsive personal portfolio website showcasing my skills, projects, internships, certifications, and professional journey as an Electronics & Communication Engineering student with a passion for Frontend Development and UI Design.

## ✨ Features

* Modern glassmorphism-inspired UI
* Dark & Light theme toggle
* Fully responsive design
* Smooth scrolling navigation
* Active section highlighting
* Animated role changer
* Project showcase section
* Internship experience section
* Certifications section
* Contact section with Email & LinkedIn integration

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts (Space Grotesk & Poppins)

## 📂 Sections

* About Me
* Skills
* Projects
* Internships
* Certifications
* Contact
